# React Learning Projects

| # | Project | Key Concepts |
|---|---------|-------------|
| 01 | [basicReact](./01basicReact) | CRA setup, simplest React component |
| 02 | [counter](./02counter) | `useState`, state batching, callback updates |
| 03 | [tailwindProps](./03tailwindProps) | Props, default props, Tailwind CSS |
| 04 | [background_changer](./04background_changer) | Declarative vs imperative UI, `useState` |
| 05 | [passwordGenerator](./05passwordGenerator) | `useState`, `useCallback`, `useEffect`, `useRef`, TypeScript |
| 06 | [todoApp](./06todoApp) | List rendering, controlled inputs, CRUD, `useEffect`, `localStorage` |
